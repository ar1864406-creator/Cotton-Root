# **App Name**: Rayyan's Digital Portfolio

## Core Features:

- Project Showcase: Display Muhammad Rayyan's projects with descriptions, tech stack, and links to live demos and GitHub repositories.
- Interactive Resume: Showcase Muhammad Rayyan's professional experience, skills, and education in an interactive format.
- Contact Form: Allow visitors to contact Muhammad Rayyan via email.
- Skills Visualizer: Visualize Muhammad Rayyan's skills using a generative AI tool that interprets skill levels and presents them in a unique graphical format. This format will update according to skill level changes.

## Style Guidelines:

- Primary color: A deep indigo (#4B0082) to reflect Rayyan's professional depth and sophistication.
- Background color: A very light gray (#F0F0F0), almost white, to ensure a clean and modern aesthetic.
- Accent color: A vibrant purple (#BE29EC) to highlight key information and interactive elements.
- Body text: 'Inter' (sans-serif) for a clean, readable experience.
- Headline text: 'Space Grotesk' (sans-serif) to give titles a techy and scientific vibe.
- Use modern, minimalist icons to represent skills, technologies, and contact information.
- Subtle animations for page transitions and interactive elements to enhance user engagement.
